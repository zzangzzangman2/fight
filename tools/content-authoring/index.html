      저장 방식

      이 화면은 JSON을 직접 보여주지 않습니다. 저장하면 내부에서 대사 목록을 Unity용 노드 데이터로 변환하고, Resources 폴더 아래 매니페스트와 업로드 에셋 경로를 갱신합니다.

      • 대사 저장 위치: UnityScaffold/Assets/JoseonMurimTactics/Resources/AuthoringContent/content_manifest.json
      • 업로드 에셋 위치: UnityScaffold/Assets/JoseonMurimTactics/Resources/AuthoringContent/Media
      • Unity 로더: AuthoringContentManifest.LoadFromResources()
      • 프롤로그와 동료 대화는 저장된 매니페스트가 있으면 그것을 우선 사용합니다.
