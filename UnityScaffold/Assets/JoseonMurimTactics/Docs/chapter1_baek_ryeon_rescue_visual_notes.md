# Chapter 1 Baek Ryeon Rescue Visual Notes

## Scene Purpose
Prepare visual hooks for the Baek Ryeon rescue beat after the first BattleTest flow. This document is planning support; story content should still be authored through the content manifest pipeline.

## Beat
The party leaves after holding the Baekdu gate. On the road toward Seorak Spear Sect, Baek Ryeon is threatened by mountain bandits. The party intervenes, then Baek Ryeon invites them to the sect and becomes a temporary ally.

## Required Visuals
- Snowy Seorak mountain road near a spear sect route.
- Adult spear heroine in white-blue clothing with a frost/snow motif.
- Rough bandit silhouettes that read as enemies without using copied designs.
- Moonlit snow forest with a few warm lantern accents.
- Optional wide event image: `Art/VisualUpgradeV1/Concepts/baek_ryeon_bandit_rescue_concept_16x9.png`.

## Integration Hooks
- MissionBoard can preview the event image after the first gate battle.
- WorldMap can use the image as a route/event card background.
- BattlePrep can use it as a contextual backdrop before Seorak Pass Rescue.

## Safety
All characters are adult, non-sexual, original Joseon murim designs. Do not bake text into the image; render text in UI.
