# Character Generation Prompt

This is the prompt used for the current first-pass Unity character art.

```text
Use case: stylized-concept
Asset type: Unity 2D character sprite source sheet for a tactics RPG
Primary request: Create a polished 2D anime visual-novel style full-body character lineup for an original Joseon murim tactics game, with the production quality and clean Unity-game asset feel of a Korean 2D visual novel. The characters must be original adult characters, not existing characters from any game or webtoon.
Scene/backdrop: perfectly flat solid #00ff00 chroma-key background for later background removal, no shadows on the background, no floor plane, no gradients, no texture.
Subject: six separate adult characters standing in a clean lineup with generous spacing: one charming adult male murim sect leader, and five beautiful adult female martial artists with elegant fashion-model silhouettes, mature proportions, expressive eyes, detailed hair, layered hanbok/murim-inspired outfits, belts, sleeves, boots, and tasteful weapon/accessory hints. Include varied hair colors/styles and distinct silhouettes: refined swordswoman, ice healer, stealth needle assassin, powerful fist fighter, charismatic strategist.
Style/medium: high-end 2D anime game character art, crisp linework, soft cel shading, polished visual novel sprite finish, clean edges for cutout use, attractive but tasteful, non-explicit.
Composition/framing: full-body front three-quarter view, all characters visible head to toe, evenly spaced in a single sheet, no overlap between characters, no text labels.
Color palette: cinematic but readable, varied outfits, no green anywhere in characters because background is #00ff00.
Constraints: all characters are adults in their 20s or older; no school uniforms; no nudity; no lingerie; no minors; no existing Random Chat characters; no logos; no watermark; no text. Keep the background exactly one flat #00ff00 color.
```

Next production pass should generate each character individually on transparent output or as layered PSD/PSB for Unity 2D Animation rigging.
