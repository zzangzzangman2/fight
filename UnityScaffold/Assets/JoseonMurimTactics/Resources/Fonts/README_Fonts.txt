Battle HUD bundled fonts

This folder contains the official MapleStory OTF fonts downloaded from Nexon's MapleStory font page:

https://maplestory.nexon.com/Media/Font

Bundled files:

- MaplestoryOTFBold.otf
- MaplestoryOTFLight.otf

Nexon license summary:

- Copyright and all rights belong to Nexon Korea Corporation.
- The font is provided free for personal and business users.
- Free use and redistribution are allowed when distributed in its original form.
- Modification/editing of the font files and selling the font itself are not allowed.
- Bundling or embedding is allowed when the copyright notice is included.
- Source attribution is recommended.

The runtime loader checks these Resources paths first:

- Fonts/MaplestoryOTFBold
- Fonts/MaplestoryOTFLight
- Fonts/MapleStory
- Fonts/Maplestory

If none exist, the game safely falls back to installed OS fonts in this order:
Maplestory OTF, MapleStory, NEXON Lv1 Gothic, Malgun Gothic, Noto Sans KR/CJK KR, Gulim, Arial.
