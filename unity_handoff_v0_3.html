Unity Handoff / v0.3

조선 무협 전술 RPG: 발더스식 파티 전술을 무협으로 변환

목표는 “바둑판 위 말싸움”이 아니라, 실제 2D 전장 위에서 동료별 무공·d20 판정·지형지물·고저차·엄폐·파훼가 맞물리는 파티 전술 전투다.

Unity 2D URPTilemap/NavNode 내부화d20 필수주행동/보조행동/반응ScriptableObjectTimeline/CinemachineFirebase AI Logic + Gemini

1. 현재 문제와 v0.3 방향

피해야 할 느낌
정사각 격자가 항상 보이고, 캐릭터가 원형 바둑알처럼 보이며, 모든 캐릭터가 비슷한 검격/장법만 쓰는 상태.
목표 느낌
전술 데이터는 타일/노드 기반이지만, 플레이어에게는 강가·누각·대나무숲·잔해·향로·등불이 있는 2D 전장으로 보인다.

2. Unity에서 쓸 기본 툴

분야추천용도
엔진Unity 2D URP2D 전장, 캐릭터 스프라이트, 파티 전술 UI.
맵Tilemap + 내부 NavNode보이는 맵은 그림처럼, 내부 이동/거리/엄폐 판정은 노드나 타일 데이터로 처리.
데이터ScriptableObject캐릭터, 무공, 지형, 상호작용 오브젝트를 데이터화.
연출Timeline + Cinemachine절기 컷인, 카메라 줌, 경공 이동, 지형 폭발 연출.
대화Yarn Spinner전투 전후 선택지, 동료 승인도, 항복 권유, 문파 이벤트.
AIFirebase AI Logic + Gemini전투 묘사, 적 대사, 자유 대사 태그화. 판정과 수치는 Unity 코드가 담당.

3. 전투 기본 규칙

항목규칙
파티박성준 + 여성 동료 4명 = 5인 파티. 적은 5명 또는 다수.
턴민첩 + d20 + 고저차 보너스로 이니셔티브 결정.
행동 경제턴마다 주행동 1, 보조행동 1, 반응 1, 이동력.
d20 공격d20 + 스탯 보정 + 숙련 + 무공 보정 + 지형/상태 보정 ≥ 대상 방어도.
기회/불리고지, 은신 암기, 파훼 대상, 물가 빙공은 기회. 강엄폐 원거리, 미끄러운 물가, 넘어짐은 불리.
무공 제한내공 비용, 쿨다운, 전투당 사용횟수, 행동 슬롯 제한.
승리살상/비살상 제압/항복/기세 붕괴/파훼 후 절기 제압.

4. 캐릭터별 무공 정체성

캐릭터역할무공전투 재미
박성준해동문 문주, 풍류검, 심리전사군자검농월추파태평농월보문주호령호색한 기믹은 노골적 성적 묘사가 아니라 도발/심리전/동료 승인도 리스크로 처리.
윤서화예검 반격수월하반조검검로재기예검반격 예약상대 공격을 읽고 반응으로 받아쳐 파훼를 쌓는다.
백련빙공, 치유, 지형 제어빙백장설화심법한설봉로물가/여울을 얼리고 둔화, 회복, 정화로 장기전을 담당.
한비연암기, 독공, 잠행비화독침흑립잠행만천화우대나무숲/연막/고지와 연계해 원거리 독공을 넣는다.
도아린권장, 밀치기, 비살상 제압파산권철산고금나수벼랑, 다리, 물가로 밀쳐 지형 피해를 만들고 항복/제압 조건을 연다.

5. 맵 설계: 바둑판 대신 2D 전장

Unity 내부 구현은 노드/타일을 써도 된다. 중요한 것은 플레이어가 정사각 바둑판을 보는 게 아니라 “압록강 폐사당”이라는 장소를 보는 것이다.

지형효과연계 무공
압록강 여울미끄러짐, 이동 방해, 빙공 강화백련 빙백장/한설봉로
누각 지붕고지, 원거리 명중 보너스한비연 암기, 적 궁수
대나무숲강엄폐, 잠행, 시야 차단한비연 흑립잠행/비화독침
객잔 술상걷어차서 엄폐 생성, 넘어짐도아린 파산권, 박성준 보법
향로연막 생성, 암기/은신 강화한비연, 윤서화 간파
등불화염 지대, 화상밀치기/도발로 적을 유도
낡은 다리파괴 가능, 여울 추락도아린 철산고/파산권
무너진 불상내공 판정으로 파편 폭발, 파훼 증가백련/박성준 내공계

6. Codex에 그대로 줄 구현 요청 요약

Unity 2D URP 프로젝트를 만든다.
전투는 5인 파티 턴제이고 d20 판정이 필수다.
화면에는 정사각 격자를 노출하지 말고, 실제 2D 전장처럼 보이게 한다.
내부적으로는 BattleNode 또는 Tilemap 데이터를 써서 이동, 거리, 고저차, 엄폐, 지형 효과를 계산한다.

필수 클래스:
- TurnManager
- ActionEconomy
- DiceRoller
- SkillResolver
- TerrainResolver
- MovementResolver
- CombatLog
- BattleCameraDirector
- TimelineCuePlayer
- GeminiNarrationClientMock

필수 ScriptableObject:
- CombatantData
- SkillData
- BattleMapData
- BattleNodeData
- InteractableObjectData

아군 5명은 모두 다른 무공 체계를 가진다.
박성준은 풍류검/심리전/문주 지휘.
윤서화는 반격검/파훼.
백련은 빙공/치유/지형 제어.
한비연은 암기/독공/잠행.
도아린은 권장/밀치기/비살상 제압.

Gemini는 판정하지 않는다.
Gemini는 전투 묘사, 적 대사, 자유 대사 intent_tag, timeline_cue만 JSON으로 반환한다.
